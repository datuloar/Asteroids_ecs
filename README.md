# Asteroids_ecs
Реализация игры asteroids при помощи leo ecs lite
## Это не тестовое задание, я работаю над этим проектом в свое удовольствие и нарабатываю свой Game Template.

 Используется :
* Zenject
* DotWeen
* UniRx
* UniTask
* ECS- Leo ecs lite
* Addressables
* MVC - UI
* Assembly definition
* ci
* Unit testing
